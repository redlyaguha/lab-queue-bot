# Lab Queue Bot

Telegram бот для управления очередями лабораторных работ.

## Функционал

- **Админы группы** могут создавать очереди (до 30 мест)
- **Студенты** сами выбирают место в очереди
- Кнопка "Лабу сдал" освобождает место
- Запрос на обмен местами с подтверждением

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Получите токен бота у [@BotFather](https://t.me/BotFather)
2. Создайте `.env` файл:

```env
BOT_TOKEN=your_bot_token
ADMIN_GROUP_ID=your_group_id
```

3. Добавьте бота в группу и сделайте его админом

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/create_queue` | Создать очередь (админ) |
| `/queues` | Список очередей |
| `/myqueues` | Ваши записи |
| `/submit_lab` | Отметить как сдавшего |
| `/swap_request` | Предложить обмен |
